1. Equality Operators (== vs ===)

Definition

== is the loose equality operator. It compares values after type conversion.
=== is the strict equality operator. It compares both value and data type. 

let a = '1';
let b = 1;

console.log(a == b);
console.log(a === b);
console.log(a != b);
console.log(a !== b);
Output:
true
false
false
true
Stepwise Explanation:
1. '1' == 1 → string converted to number → true
2. '1' === 1 → type mismatch → false
3. != follows == rules → false
4. !== follows === rules → true
Interview Tip: Use === to avoid unexpected type conversion bugs.

2. Unary + and Post Decrement

let a = '1';
let b = 1;

let res1 = +a + b--;
console.log(res1);

let res = a + b;
console.log(res);
Output:
2
11
Stepwise Reasoning:
+ '1' converts string to number → 1
1 + 1 = 2
b-- decreases value after use → b becomes 0
'a' + b → string concatenation → "11"

3. Boolean Conversion Rules

let a1 = '0';
let a = Boolean(a1);

let b1 = 0;
let b = Boolean(b1);

console.log(a == b);
console.log(a === b);
Output:
true
true
Non-empty strings → true
0 → false
Both variables become boolean false → equal and strictly equal

4. null vs undefined

console.log(null == undefined);
console.log(null === undefined);
console.log(null >= 0);
console.log(null > 0);
Output:
true
false
true
false
null loosely equals undefined
null converts to 0 in numeric comparison
0 >= 0 → true

5. NaN (Not a Number)

console.log(NaN == NaN);
console.log(isNaN(NaN));
console.log(isNaN('hello'));
Output:
false
true
true
NaN is not equal to any value, including itself.
Always use isNaN() to detect NaN.

6. String Comparison (Unicode)

let str4 = 'goms';
let bool4 = 'Zoms';

console.log(str4 > bool4);
Output:
false
'g' → Unicode 103
'Z' → Unicode 90
Comparison is lexicographical (character by character).
