JavaScript Type Casting & Conversion

1. What is Type Casting?

Type casting (or conversion) is changing a value from one data type to another. JavaScript uses 2 types:
  • Implicit Conversion: JS changes type automatically
  • Explicit Conversion: Developer changes type manually
JavaScript is dynamically typed, so types can change at runtime.

2. Common Conversion Functions

  • String() → convert to string
  • Number() → convert to number
  • Boolean() → convert to boolean
  • parseInt() → convert to integer
  • parseFloat() → convert to decimal
Note: parseInt() and parseFloat() ignore characters and extract numbers.

4. Truthy & Falsy Values

JavaScript converts values to true or false when needed. Falsy Values:
  • false
  • 0
  • "" (empty string)
  • null
  • undefined
  • NaN
Everything else is truthy!

5. Why Type Conversion Happens?

JavaScript converts values automatically during:
  • Math operations
  • Comparison
  • Boolean checks
  • String concatenation
Example: "12" / 6 ➝ 2 (String converted to Number automatically)
