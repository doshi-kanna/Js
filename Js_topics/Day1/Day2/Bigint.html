1. BigInt

JavaScript normally uses Number type to store numeric values, but it has a limit. The largest safe integer is: 2^53 - 1. When we need to store larger numbers, we use BigInt.

BigInt values end with n.

let bigValue = 90000202020n;
console.log(bigValue + 1n);

BigInt is useful when:

  • Working with very large numbers
  • Cryptography
  • Financial calculations

2. Strings and Template Literals

JavaScript has three ways to write strings:

  • 'single quotes'
  • "double quotes"
  • `backticks` (template literals)

Template literals allow embedding values inside strings:

let name = "Tony";
let message = `Hello, my name is ${name}`;
console.log(message);

Advantages of template literals:

  • Supports multi-line text
  • Supports variable embedding
  • Easy to read and write

3. JavaScript Data Types

JavaScript has two kinds of data types:

Primitive Types

  • Number
  • BigInt
  • String
  • Boolean
  • Null
  • Undefined
  • Symbol

Non-Primitive Type

  • Object (arrays, functions, dates, etc.)
